# Swarm Learning

This directory contains examples of [Swarm Learning](https://nvflare.readthedocs.io/en/main/apidocs/nvflare.app_common.ccwf.html) with NVIDIA FLARE — a decentralized federated workflow where clients act as both trainers and aggregators in a peer-to-peer fashion.

## Examples

- **[swarm_pt](./swarm_pt/)** — Swarm learning with LoRA fine-tuning: fine-tune a Qwen2.5 language model using LoRA adapters in a federated setting. See the [swarm_pt README](./swarm_pt/README.md) for setup and run instructions.
