User Guide - Provision, Start, OperateÂ¶

This user guide shows how to use NVIDIA FLARE to deploy and operate an FL system on multiple sites with a Provision-Start-Operating procedure. A reference application will be used here to show provisioning and basic operation of the system through the admin client, but for more details on what you can do with apps with custom components and the flexibility that the new NVIDIA FLARE APIs bring, see the Programming Guide - Developing Apps with NVIDIA FLARE.

You can also see some example applications integrating with Clara Train and MONAI to see the capabilities of the system and how it can be operated.
